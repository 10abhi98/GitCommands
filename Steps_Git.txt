## Steps to add file or project to GITHUB using cmd!!!

**Before Starting we need to set username and email id using "git config":

$ git config : This command sets the author name and email address respectively to be used with your commits.

		Usage: git config --global user.name “[name]”  
			   git config --global user.name "abc"


		Usage: git config --global user.email “[email address]”
			   git config --global user.emali "abc@gmail.com"

$ git status: This command discards all history and goes back to the specified commit.

	   Usage: git status
================================================================================================================
=======

1) git init: It creates a local repository in the given folder from where we need to upload file or project onto GitHub.	
			 It generates a file as ".git" which is by default hidden.
			 SYNTAX: git init

2) git add:  Allows you to upload file from a given folder to GitHub.
			 You can add single or multiple Files.
			 SYNTAX: git add . (add all files in the folder)
			 		 git add [filename] (adds single file)

3) git commit: This command records or snapshots the file permanently in the version history.
			   git commit -m “[ Type in the commit message]”

4) git remote: This command is used to connect your local repository to the remote server.
			   git remote add [variable name] [Remote Server Link]
			   

**** If there are already file in repository then we first need to pull those files and then push it.

5) git pull:This command fetches and merges changes on the remote server to your working directory
			
			**Try Either of these commands if it shows error!!
			git pull [Repository Link]
			OR
			git pull 
			OR
			git pull origin master
			OR
			git pull origin master --allow-unrelated-histories


6) git push: This command sends the committed changes of master branch to your remote repository.
			 git push [variable name] master(The main branch is called master)
